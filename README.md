# Customer Segmentation & Profiling

## Overview

This project focuses on **Customer Segmentation and Profiling** using Machine Learning techniques. The goal is to group customers based on behavior and characteristics to better understand customer needs and support targeted business strategies.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib / Seaborn

## Key Steps

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Feature selection and transformation
* Customer segmentation using clustering (K-Means)
* Profiling customer groups based on behavior

## Outcome

Successfully segmented customers into meaningful groups and created customer profiles that help in improving marketing strategies, personalization, and business decision-making.
